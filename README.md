# NLW4-Reactive
Move IT App

#Installation  
https://nodejs.org/en/download/package-manager/#macos  

# Create Project  
`yarn create react-app moveit --template=typescript`  


# Create Next Project  
`yarn create next-app moveit-next`  


# Install complements  
`yarn add typescript @types/react @types/react-dom @types/node -D`  


# Start Server  
`yarn dev`
